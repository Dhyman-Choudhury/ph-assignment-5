Question-1: what are the different ways to select an element in DOM?

There are several different ways to select an element in DOM.
1. getElementById 2. getElementsByClassName 3.getElementsByName
4. getElementsByTagName 5. querySelector 6. querySelectorAll

Question-2: What is the different between innerHTML, innerText, and textContent?

1. innerHTML set or get HTML content.It can read HTML and shows hidden text.
2. innerText cannot read html and cannot show hidden text. It shows only visible text.
3. textContent cannot read html but shows hidden text. It shows all text

Question-3: What is event delegation in the DOM?

Event delegation is a technique where instead of adding event listeners to
multiple child elements, you add a single event listener to a parent element.
The event then "bubbles up" from the child to the parent, allowing the parent to handle it.

Question-4: What is event building in the DOM?

Event binding is the technique of connecting an event (like click, mouseover, keyup, keydown etc.)
with an event handler for an element in the DOM. It is also called event handler.
